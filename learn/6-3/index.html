⬡ STATION 01 — THEORY + PRACTICE

Data Systems

Master validation, verification, error types, data wrangling and GIGO.

VALIDATION
✅
Automated Format Check
An automated check that data meets pre-defined rules for type, format, range or length. Validation does NOT confirm factual accuracy — only that data is in the right shape. Examples: presence check, format check, range check, type check.
KEY POINTA valid postcode format does not mean the postcode is real — only that it looks right
VERIFICATION
🔍
Accuracy Check Against Source
A process that confirms data is accurate by comparing it against an authoritative source or using double-entry. Verification checks truth, not format. Methods: double-entry (two operators, compare results), visual check (compare against original document).
KEY POINT07700 900001 vs 07700 900100 — passes validation, fails verification
ERROR TYPES
⚠️
Transcription vs Transposition
Transcription error: wrong value entered entirely (1999 → 2199). Transposition error: adjacent characters swapped (51 → 15). Both pass validation checks. Only verification can catch them. Always name the specific type in exam answers — "entered incorrectly" scores zero.
EXAMINER TIPName the error type precisely — the specific term earns the mark
DATA WRANGLING + GIGO
🔧
Clean Data, Quality Output
Data wrangling: cleaning, transforming and mapping raw messy data into a usable format — includes deduplication, standardisation and normalisation. GIGO (Garbage In, Garbage Out): no matter how sophisticated the system, poor input data produces poor output. Link GIGO to a specific downstream failure in exam answers.
GIGO EXAMPLE12% address errors → 12% incorrect delivery routes, regardless of routing system quality

DEFINITION

TERM

DEFINITION
A process that confirms data accuracy by comparing it against an authoritative source or through double-entry to detect discrepancies.
TERM
Transcription Error
DEFINITION
An error where the wrong value is entered entirely — e.g. DOB recorded as 19/04/2199 instead of 19/04/1999. Passes validation.
TERM
Transposition Error
DEFINITION
An error where two adjacent characters or digits are swapped — e.g. sort code 20-51-63 entered as 20-15-63. Passes validation.
TERM
Data Wrangling
DEFINITION
Transforming and mapping raw, messy data into a structured, clean format — including deduplication, standardising formats and resolving inconsistencies.
TERM
GIGO
DEFINITION
Garbage In, Garbage Out — poor quality input data always produces poor quality output, regardless of how sophisticated the processing system is.

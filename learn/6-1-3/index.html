⬡ STATION 01 — THEORY + PRACTICE

How Data Is Generated

Master H2D vs D2D, IoT, and the data-to-information transformation.

HUMAN-TO-DEVICE (H2D)
🧑‍💻
Human Actively Inputs Data
A human physically types, clicks or speaks data into a device. The generation requires human action at the point of capture. Examples: nurse typing observations, customer entering a postcode, receptionist logging arrivals.
KEY TESTIs a human performing the input action? If yes — H2D.
DEVICE-TO-DEVICE (D2D)
📡
Devices Exchange Data Automatically
Devices communicate and transfer data without any human action at the point of capture. Examples: RFID tags updating stock systems, GPS trackers transmitting location, smart meters sending energy readings.
KEY TESTDo devices exchange data without human input? If yes — D2D.
INTERNET OF THINGS (IoT)
🌍
Connected Sensor Networks
A network of internet-connected physical devices that collect and transmit data continuously in real time. IoT is always D2D. Examples: warehouse temperature sensors, factory vibration monitors, retail footfall trackers.
ADVANTAGEContinuous, real-time data at scale — impossible with H2D
DATA → INFORMATION
🔄
The Transformation
Raw generated data only becomes information when processed and given context. 38.7 is data. "Patient temperature 38.7°C — above normal range at 14:23" is information. Requires: structure + context + meaning.
EXAMINER NOTEThe generation method (H2D/D2D) does not determine whether something is data or information

TERM
Human-to-Device (H2D)
DEFINITION
Data generation where a human actively inputs data into a device at the point of capture — e.g. typing, clicking, scanning manually.
TERM
Device-to-Device (D2D)
DEFINITION
Automated data exchange between devices without human intervention at the point of capture — includes IoT, RFID, GPS trackers and smart meters.
TERM
Internet of Things (IoT)
DEFINITION
A network of internet-connected physical devices that collect and transmit data continuously in real time without human action — always D2D.
TERM
Data to Information
DEFINITION
Raw data becomes information when processed and given structure, context and meaning. The generation method does not affect this classification.
COMPARE
H2D Advantage
ANSWER
Lower setup cost, flexible, suitable for low-volume or irregular data. Best for small organisations or data types that cannot be automated.
COMPARE
D2D Advantage
ANSWER
Fast, continuous, scalable, reduces human error, operates 24/7. Best for high-volume, time-sensitive data where human input would be too slow or expensive.
